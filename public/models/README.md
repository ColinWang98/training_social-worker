# Local Avatar Models

VRM/GLB avatar assets are intentionally not committed to this public repository.

Place local prototype assets here when running the app, for example:

- `client-john-do.vrm`
- `client-john-do-arkit.vrm`
- `mobf005.vrm`
- `client-haru.glb`

Some models used during local testing have embedded credit or redistribution
requirements. Keep those files private unless their license allows public
redistribution.
